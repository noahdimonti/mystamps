# MyStamps Backend

Backend service for **MyStamps**, a loyalty and rewards application.

This project is currently under active development and serves as the API layer responsible for business logic, data persistence, and authentication. The frontend client(s) (e.g. web or mobile) will communicate with this service via HTTP.

## Tech Stack (initial)
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
